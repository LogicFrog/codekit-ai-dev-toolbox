CodeKit-AI-DevToolbox
A lightweight, extensible LLM-driven intelligent code assistant designed for individual developers. Simplify your coding workflow, boost efficiency, and lay a solid foundation for AI-enhanced development.
✨ Core Features

- Local code management with efficient CRUD operations for code snippets

- Integrated keyword + RAG semantic search for quick code retrieval

- Lightweight Git version control integration

- Standardized AI Agent access layer (ready for LLM integration)

- Stable base framework built with Spring Boot 3 & Vue3

🔧 Tech Stack

Backend: Spring Boot 3 | Frontend: Vue3 | Git | LLM Integration Ready

🚀 Getting Started

1. Clone the repository

git clone https://github.com/LogicFrog/codekit-ai-dev-toolbox.git

2. Import the project into your IDE (IntelliJ IDEA recommended)

3. Configure dependencies and run the backend service

4. Launch the frontend and start using the core features
